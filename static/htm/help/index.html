前言

前两天看了 吉米·威尔士关于维基百科诞生的演讲 ,再一次被开源的力量所震撼 ,我觉得, 我应该也去搞一个NB的开源项目

因为发现很多初学者用python写网站没经验, 写出来的网站比较卡, 让我很痛心,于是这一次, 我又开始了一个坑 ,打算基于Python的Tornado, 打造一套可以抗千万PV的SNS/微博/轻博客的开源基础框架

让以后Python程序员写SNS, 省去很重复开发的工作 - 比如登录, 比如头像, 比如相册, 比如日记, 比如等等等... ,现在项目从零开始, 欢迎参与, 前端, 美工, 程序员, 都可以!
